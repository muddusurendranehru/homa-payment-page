## 📋 **Deployment Steps**

### **Simple Backend:**
```bash
1. Upload simple-backend/ folder to Replit
2. Add 3 secrets (DATABASE_URL, JWT_SECRET, PORT)
3. Click Run ▶️
4. Done! Test immediately
```

