================================================
📦 UPLOAD METHOD 1: ZIP FILE
================================================

1. Select all folders listed above
2. Create ZIP file: homa-payment-system.zip
3. In Replit, click "Upload" and select ZIP
4. Extract in Replit workspace

================================================
📦 UPLOAD METHOD 2: GITHUB (RECOMMENDED)
================================================

1. Push your project to GitHub
2. In Replit, click "+ Create Repl"
3. Choose "Import from GitHub"
4. Paste repository URL
5. Click "Import"

================================================
🔧 AFTER UPLOAD:
================================================

1. Add all 13 environment variables to Secrets 🔒
2. Click Run ▶️
3. Wait for "npm install" to complete
4. Server starts automatically
